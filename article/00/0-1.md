# はじめに

この書籍では、これから WordPress を触り始める方向けに、WordPress の基本的な使い方を解説しています。

## 主な対象とする読者

本書は以下のような読者に向けて執筆をしました。

- WordPress を初めて触る方
- これからブログやウェブサイトを作成しようと考えている方
- WordPress の基本的な操作方法を学びたい方
- ブロックエディターの基本的な使い方を理解したい方
- WordPress の管理画面の各機能について知りたい方
- WordPress を使ってコンテンツを作成・管理する方法を学びたい方
- WordPress の基本的な設定方法を理解したい方

## 本書では扱わない内容

本書では、以下のような内容については詳細を取り扱いません。

- WordPress の詳しいインストール方法
- テーマやプラグインの開発方法
- WordPress の高度なカスタマイズ方法
- 標準テーマ以外の特定のテーマに関する情報
- 個人レベルで取り組むことが難しいホームページ運営方法

これらについては本書では詳しく取り扱いません。必要に応じて、WordPress 公式ドキュメントや他の専門書籍を参照してください。

## なぜ本書を執筆しようと思ったのか

これまで WordPress 標準テーマを基軸とした書籍を複数出版してきました。初めて出版した当時は WordPress のバージョンは 6.4 で、標準テーマは Twenty Twenty-Four だったと記憶しています。

それから複数書籍の出版を経て、WordPress 6.7 に対応した『WordPress 6.7 対応 Twenty Twenty-Five でウェブサイトを作ろう』を出版し、多くの方の目に触れる機会を得て、さまざまなご意見やご感想をいただきました。

その中でも多くいただけた感想として、これから WordPress を始めようとしている方のお役に立てたというものが多くありました。これまでの書籍全てにおいて、これから WordPress を触り始めようとしている方の役に立てる書籍にすることを強く考えて執筆をしてきた手前、とても嬉しく感じたことを記憶しています。

その一方で、WordPress のバージョンアップにともない、書籍内容を再編成したりデモサイトの内容を考え直したりすることは簡単ではなく、本書ではできるだけバージョンアップに伴う改訂が行いやすい構成に変更を進めています。

現在、本屋さんのホームページ制作コーナーに立ち寄っても、WordPress の入門的な内容を扱う書籍はあれど、特定のテーマを使った使い方や作り方を紹介するものや、そもそも最新のバージョンには対応していないものも多く見受けられます。

そういった書籍を手に取ってしっくりこなかった方に向けて、WordPress の基礎的な内容を学習できて、どんなテーマやプラグインを使用・併用したとしても応用が効く、そんな内容の書籍にするべく執筆をしました。

本書が、今後みなさんが WordPress を楽しまれるためにも、WordPress の基本的な機能を理解し、標準テーマを通した振る舞いに触れ、さまざまなプラグインや他のテーマを活用した発展的なウェブサイト制作に繋げていただくことに役立てば本望です。

## 執筆するにあたり意識したこと

本書では、できるだけ丁寧な説明を心がけました。ですので、本書で想定している読者であれば内容を参照し、その通りに作業を進めることで多くのことを理解いただけるように構成しています。

サイトエディターやブロックエディターを見たことがない、またはあまり触れたことがない方には、少し難しく感じられる点もあるかもしれませんが、少しずつ触っていくことで理解が深まることは間違いありません。ぜひ、触ってみながら理解を深めていってください。

本書を通して、WordPress やサイトエディター、ブロックエディターへの理解が少しでも深まれば幸いです。

## 理解を深められる読み進め方

すでに WordPress を触っている方であれば、本書に掲載しているスクリーンショットを見るだけで想像できることが多いでしょう。しかし、これから WordPress を触り始めるという方にとって、多くははじめて見る画面でしょう。ですので、実際に WordPress を触りながら本書を読み進めることをオススメします。

以下に手軽に試せる環境を紹介します。より詳しい方法はインターネットで検索していただくか、利用レンタルサーバー会社にお問い合わせいただくと良いでしょう。また、対面で WordPress の利用環境について質問・相談したい場合には、各地域の WordPress Meetup に参加してみるのも良いでしょう。

本書では下で紹介する WordPress Playground を活用した環境を用意しています。詳しくは下記サポートサイトを参照してください。

### WordPress Playground

ブラウザで WordPress を動かせるサービスです。手軽に WordPress を試せますが、ブラウザをリロード（更新）すると全ての設定がクリアになります。ローカル等に設定を保存する機能もありますが、日本語でわかりやすく紹介する情報はあまりありませんので、各自の責任に利用してください。

- `https://wordpress.org/playground/`

### Local by Flywheel または Studio by WordPress.com

それぞれのアプリケーションをインストールし、PC の中に WordPress 環境を用意できます。利用方法を紹介する情報も多いので、参考にしながら試してみるのも良いでしょう。

- `https://localwp.com/
- `https://developer.wordpress.com/studio/`

### ご自身が契約しているレンタルサーバー上

ご自身が契約されているレンタルサーバー上に、自由に使って壊しても良い WordPress 環境を用意して試すこともできます。
