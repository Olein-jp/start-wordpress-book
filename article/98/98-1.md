# おわりに

WordPress 6.9 をベースに、管理画面で可能な設定内容からブロックエディター、サイトエディターの各メニューの紹介、そしてよく利用することが多いコアブロックでできることと活用方法、最後に簡単なウェブサイトを作ってみるという流れで本書を進めてきました。

標準テーマである Twenty Twenty-Five を使用して説明をしてきましたが、私は WordPress の基本を理解・習得するために最適なテーマは標準テーマだと考えています。ですので、これから WordPress を使い始める読者の皆さんは、まずは標準テーマを利用して WordPress でどんなことができるのかを確かめていただくと良いと思います。

その上で、世界中にたくさんある様々なテーマを利用すると、より便利なポイントが明確になったり、何かわからないことやトラブルが発生した際にも、テーマが原因なのか WordPress 本体に関係することなのかを判断できるようになるでしょう。

近年の WordPress のアップデートでは、多くの点がブロックテーマを視野にいれた改善や機能追加となってきていると考えています。ですので、これから WordPress を使い始めるという方々には、ぜひブロックテーマも触っていただきたいと考えています。

近年の WordPress の変化は激しく、印刷書籍での情報提供が難しいと個人的には考えています。執筆・編集・印刷などの工程を経ている間に次のメジャーアップデートを迎えてしまう可能性もあるため、新しいバージョンに対応した情報源がなかなか存在しないのも事実でしょう。本書はそのような点も考慮しつつ、将来訪れるであろうメジャーアップデートにも対応していけるようなメンテナンス性も確保した出版スタイルを取っています。

よって、誌面レイアウト等になかなかこだわれないという側面もあるのですが、できるだけ新しい情報を**これから WordPress を触り始めようという方に向けて届けられるよう**に努力しています。ご理解を頂けますと幸いです。また、個人出版という点もあり、**多くの皆様の手に渡ることが次のバージョンに対応した書籍を改訂出版する後押しにもなります。よろしければ、皆さんの周りの方々にも本書をご紹介いただけますと幸いです。**

本書がこれまで・これから WordPress を使う方々のお役に、少しでも立っていれば嬉しいです。読んでいただきありがとうございました。
